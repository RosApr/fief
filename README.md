# fief

## 🚀 Quickstart

```bash
deno init .
deno add jsr:@fre/kit
```

Create a simple page in `app/routes/index.tsx`

```tsx
export default function Home() {
  return <div>this is homepage</div>;
}
```

Run the dev server

```bash
deno task start:dev
```

Open [http://localhost:8000 🎉](http://localhost:8000)
